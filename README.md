# Movie Recommendation System

## Description
This project is a simple Movie Recommendation System developed in Python using the K-Nearest Neighbors (KNN) algorithm to suggest movies based on user preferences.

## Features
- Recommends movies based on user interests
- Simple and easy-to-use interface
- Implemented using Python
- Uses the KNN algorithm

## Technologies Used
- Python

## Available Movies
- Avatar
- Titanic
- Inception
- Interstellar
- Iron Man
- Avengers Endgame
- Captain America
- John Wick
- Frozen
- Toy Story

## How to Run
1. Open the Python file.
2. Run the program.
3. Enter the required input.
4. View the recommended movies.
